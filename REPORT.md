# QLD Alpha Rotation 보고서

발행일: 2026-05-17

이번 버전은 고정 전략 선택이 아니라, 월간 리밸런싱을 기본으로 두고 월중 손실 제한과 회복 재진입을 붙인 펀드매니저형 운용 구조다. 목표는 단순 QLD 보유보다 높은 수익률을 유지하면서 최대 낙폭을 낮추는 것이다.

## 핵심 결과

검증 기준: Yahoo Finance 일봉, 워크포워드 방식, 2018-06~2026-05.

| 2018-06~2026-05 MDD | -20.5% | -62.0% | +41.5%p 개선 |

| 월간 승률 | 60.8% | - | - |
| QLD 월간 초과율 | 49.5% | - | - |
| 2019-05~2019-07 케이스 | -9.7% | +0.1% | 5월 급락 손실을 기존보다 축소 |

## 전략 구조

1. 월초에 QQQ 추세, MFI, RSI, MACD, 63일 변동성, 126일 낙폭, 상대강도를 보고 목표 비중을 만든다.
2. QLD/QQQ를 코어로 두고 SOXL, TQQQ, USD, SMH, SOXX, XLK, VGT, NVDA, AVGO, AMD, MU, TSM, ASML, MSFT, META, GOOGL, AMZN을 후보로 평가한다.
3. 3배 레버리지는 최대 30%, 2배 레버리지는 최대 60%, 일반 개별주/ETF는 최대 30%로 제한한다.
4. 월중에는 QQQ/반도체 추세와 전략 자체의 평가손실을 감시한다.
5. 손실 제한이 발동하면 위험자산을 BIL/SHY 쪽으로 옮기고, QQQ 5일 반등·MFI 재유입·MACD 개선이 겹치면 부분 재진입한다.

| QQQ 5일 반등 + MFI 재유입 + MACD 개선 | QLD/QQQ/SMH 중심으로 부분 재진입 |

## 사용 방법

리플레이 탭의 월간 리밸런싱 표에서 특정 월 행을 누르면, 위의 누적식 리플레이 차트와 기간 수익 계산이 그 월로 바로 이동한다. 긴 기간을 보고 싶으면 투입 시작일과 보유 중단일을 직접 바꾸면 된다.

실전 운용 탭은 `종목,수량,평가액` 형식의 보유내역을 저장해 실제 현재 비중과 전략 목표 비중을 비교한다. 수량이 있으면 최신 가격으로 평가하고, 수량이 비어 있거나 0이면 입력한 평가액을 사용한다. 월초 기준 저장을 누르면 월중 손실 제한이 그 기준 평가액 대비로 계산된다. 실시간 알람은 페이지가 열려 있고 브라우저 알림 권한이 허용된 동안 5분 간격으로 손실 제한, 지표 감산, 회복 재진입 시점과 규모를 점검한다.
